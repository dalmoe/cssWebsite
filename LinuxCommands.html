Unix Command Cheat Sheet

Here is a list of common Linux/Unix commands. For a more complete list of commands, click here.

Common Commands

Command Description Usage
cd Changes working directory to the one specified. cd Workspace
ls Lists files in the current directory. ls Workspace
sudo Gives root permissions. sudo apt-get install
apt-get install Allows you to install a program from a repository sudo apt-get install git
mkdir Creates a new directory in the current directory mkdir newFolder
tree Displays a directory tree for the currnet directory tree Workspace
man Shows the manual for a command. man git
history Shows a history of past used commands history
